Why use ImageBox?

  1. First, you cannot drag the image (to the url bar) because you set the background-image to the element.
  2. It is good to hide the image so that people don't easily get the image (surely it still can be done by inspect element on your web page).
  3. Cannot save image as..
  4. If you inspect, you'll see the element is wrap up with padding-top
  5. Lastly, it's responsive.
